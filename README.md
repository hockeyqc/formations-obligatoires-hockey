# Formations obligatoires — Hockey Québec

Formulaire interactif permettant d’identifier les formations obligatoires selon la région, le parcours, la division, la classe et le rôle sélectionnés.

Le site est conçu pour être intégré dans une page Spordle au moyen d’une iframe.

## Publication

Le contenu du dossier `site` est publié automatiquement sur GitHub Pages lors de chaque modification de la branche `main`.
